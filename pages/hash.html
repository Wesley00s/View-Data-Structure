HASH TABLES

Uma hash table é uma estrutura de dados que utiliza uma função de hash para mapear chaves a posições específicas em um conjunto de "buckets" ou "slots". A função de hash transforma a chave em um valor numérico, determinando assim a localização do "bucket" onde o valor associado à chave será armazenado. Ao adicionar um par chave-valor, a função de hash é aplicada à chave, indicando o "bucket" correspondente. Durante a recuperação de um valor, a mesma função de hash é empregada, direcionando imediatamente para o "bucket" apropriado, evitando a necessidade de percorrer toda a estrutura.
