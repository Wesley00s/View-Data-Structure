ÁRVORES

As árvores representam um dos tipos de estruturas de dados mais importantes da computação. A estrutura de uma árvore define uma organização de elementos dispostos de forma hierárquica. Os nós de uma árvore representam espaços onde podem ser armazenados diversos tipos de informações, As arestas representam a forma com a qual os nós se conectam, e essa relação depende totalmente da forma como a árvore é utilizada e do tipo de informação que os nós da árvore armazenam. Pesquisar na web é como explorar uma árvore de conhecimento. O ponto de partida é o tópico geral, e cada busca é um caminho pelos ramos, indo de informações amplas para detalhes específicos. Cada clique aprofunda a jornada pela vastidão da internet. Experimente buscar sobre tecnologia e cultura pop.
